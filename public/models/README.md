# Model Assets

These files are third-party/open anatomy assets used by Open Anatomy Atlas.

- `z-anatomy/`: Z-Anatomy / BodyParts3D model converted to GLB.
- `open3dmodel/`: AnatomyTOOL Open3DModel fallback selection models.

The model assets retain their original Creative Commons Attribution-ShareAlike license terms. See [ASSET_LICENSES.md](../../ASSET_LICENSES.md) for attribution.
